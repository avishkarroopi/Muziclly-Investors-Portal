What's Inside M-Lab

Virtual Piano

Full 88-key piano with realistic sound sampling. Supports touch, mouse, and MIDI keyboard input. Includes chord library and scale overlay for learning.

Chromatic Tuner

Real-time pitch detection using device microphone. Shows note name, cents deviation, and visual tuning needle. Pre-configured for all common instruments.

Smart Metronome

BPM range 20-300. Time signatures from 2/4 to 12/8. Subdivision options, accent patterns, and tempo tap feature. Silent mode with visual flash.
